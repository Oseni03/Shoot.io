## What to build

Implement the Shoot feature: a Chrome extension content script that injects a button into Indeed's apply modal, scrapes the job description, calls the API to tailor the user's master resume via AI, and auto-fills the application form fields.

This is the feature that makes the product real. See the full PRD for user stories, decisions, and module breakdown.
